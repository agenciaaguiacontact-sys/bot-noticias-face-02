# 📰 Facebook News Bot 02 - Reels Edition

Este é um bot profissional de notícias automatizado para Facebook, configurado para gerar e publicar **Reels** de alto impacto.

## 🚀 Funcionalidades
- **Geração de Reels**: Transforma notícias em vídeos de 10 segundos (1:1).
- **Áudio Dinâmico**: Seleção aleatória de trilhas sonoras da pasta `AUDIOS NEWS`.
- **IA Gemini**: Resumos de notícias com ganchos virais.
- **Automação 24h**: Roda via GitHub Actions a cada 10 minutos.

## 💡 Sugestões de Nichos
1. **Notícia no Bolso**: Utilidade pública, auxílios e economia.
2. **Brasil Curioso**: Fatos bizarros e notícias inusitadas.
3. **Resumo da Fé**: Atualidades do mundo cristão e milagres.
4. **Saúde e Longevidade**: Dicas de bem-estar e notícias científicas.

## 🛠️ Configuração
O bot utiliza as seguintes variáveis de ambiente (Secrets):
- `FB_PAGE_ID`, `FB_TOKEN`, `FB_APP_ID`, `FB_APP_SECRET`, `GEMINI_API_KEY`, `SFY_EMAIL`, `SFY_PASSWORD`.
